# Proyecto: Calculadora de Interés Simple

Este proyecto contiene un script en Bash que calcula el interés simple basado en los valores ingresados por el usuario.

## Calculadora de Interés Simple
Esta calculadora permite computar el interés simple utilizando la fórmula:

**Interés Simple = (P × R × T) / 100**

Donde:
- **P** = Capital principal (Principal amount)
- **R** = Tasa de interés anual (Annual interest rate)
- **T** = Período de tiempo en años (Time period in years)

## Cómo usar
1. Ejecuta el script `simple-interest.sh`
2. Ingresa el capital principal
3. Ingresa la tasa de interés
4. Ingresa el período de tiempo
5. El script mostrará el interés simple calculado

## Ejemplo
Si ingresas:
- Capital: 1000
- Tasa: 5%
- Tiempo: 2 años

Resultado: Interés Simple = 100
